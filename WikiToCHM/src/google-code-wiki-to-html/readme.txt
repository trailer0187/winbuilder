Required 
	JAVA (probably the JDK) v5.0, download from http://java.sun.com
		Set JAVA_HOME variable in Windows

full command line arguments:

		cmd: google_code_wiki_download <project_name> [download_folder] [download_resource_extensions]

Added (by Winbuilder team) a parameter to control content, and to also remove some items.

		cmd: google_code_wiki_download <project_name> [download_folder] [download_resource_extensions] [content to show]
